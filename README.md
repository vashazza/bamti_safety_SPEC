# bamti_safety_SPEC
Safety SPEC for LLM Fine Tuning Evolved with Alphavolve

## Code
- `alpha_elo.py`: Single (sentence-specific) SPEC alpha evolve code with EloRating
- `group_ev.py`: Group Evolutionary code evaluation of connection factors between SPEC sentences

## Results
- `seed_SPEC/`: Prior to evolution, seed SPEC simply generated by receiving constitution, domain, and task instructions with Claud-3.7-sonnet
- `SPEC_final/`: Final SPEC that has evolved.
